Portfolio block. Showcase your projects and portfolio work.

== Description ==

A portfolio block that lets you use images grid with multiple styling options. 

The block displays a grid of images with options to link them to the work you present. 

You can control the layout, the global colors, item background image, item overlay box-shadow css rule, item title level, animate portfolio loading, animate items image, image sliding strip on hover, image sliding strip Width and more. 

You can export\import the portfolio attributes to a json file. This gives you an option to make changes without worrying if you don't like the outcome, since you can always load back the attributes backup. It also gives you the option to import the attributes to other occurrences of the block in the same or another site.

It may be added to any page or post using the WordPress 5+ Gutenberg editor.

= Features =

- **Customizable**: The block has many options to control the look of the portfolio.

1. It includes a parent block, the portfolio main block, and an item child block.
2. Each portfolio item content is controlled separately
3. The style is controlled globally (per portfolio block, not throughout the website)
4. you can export/import the settings of the portfolio


= How to use =
- Install ATR Portfolio plugin
- Add ATR Portfolio from the block category "Widgets" in the editor.
- Use the + sign **inside** to add items to the portfolio. Use the settings sidebar to set the item image etc.
- Control the block settings from the settings sidebar for global settings.
- Enjoy!

== Installation ==

Installing "Atr Portfolio" can be done either by searching for "Atr Portfolio" via the "Plugins > Add New" screen in your WordPress dashboard, or in the components search box in the editor.

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **ATR Portfolio**
3. Click on the **ATR Portfolio** to add the block

Or by using the following steps:

1. Upload the plugin files to the `/wp-content/plugins/atr-portfolio` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
